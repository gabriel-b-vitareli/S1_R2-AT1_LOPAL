# Atividades LOPAL (Lógica de Programação) com Flowgorithm


## 👾 Tecnologias Usadas

Esse projeto foi desenvolvido com as seguintes tecnologias:

- [Flowgorithm](http://www.flowgorithm.org/download/index.html);
- Git;
- GIthub;

## 💻 Projeto

Foram realizadas cinco tarefas sobre lógica de programação utilizando o software Flowgorithm, afim de resolver problemas de raciocínio lógico, construção de passo-a-passo e estudos sobre a utilização de git e github.

##

## 👤 Atividade 1:

<img width="701" height="693" alt="image" src="https://github.com/user-attachments/assets/20e1fa55-f045-4c70-917d-c4e7504f987e" />

Essa atividade consiste em um algoritmo que solicita ao usuário seu nome, e em seguida o cumprimenta. 
Primeiro, definimos a variável "nome", constituiída apenas por caracteres. Depois, o computador envia a mensagem "Digite seu nome:" para indicar ao usuário que ele deve digitar seu nome.
Após isso, o usuário pode digitar seu nome por meio da função de entrada, dessa forma, o que o usuário digitar será o novo valor da variável "nome". Por fim, o computador cumprimenta o usuário a partir do bloco de saídas com o seguinte "código":
"Olá, " & nome&"!"

Assim, se for digitado, por exemplo, "Gabriel", o computador dirá: "Olá, Gabriel!"

## 🧮 Atividade 2:

<img width="431" height="697" alt="image" src="https://github.com/user-attachments/assets/dbce0c59-1b13-43fb-8cfc-4baa016461bb" />

Nessa segunda atividade, foi criado um algoritmo que soma dois números reais de escolha do usuário. 
Primeiro, definimos duas variáveis, "primeiroNumero" e "segundoNumero". Em seguida, o computador solicita (por meio de uma mensagem de saída) que o usuário digite o primeiro número, e um comando de entrada permite o usuário digitar qualquer número real, assim definindo o valor da variável. Em seguida, ele faz o mesmo processo para o segundo número.
Por fim, utilizamos um bloco de saída com o seguinte código: "Soma realizada! O resultado foi: " & primeiroNumero + segundoNumero
Assim, ele manda a mensagem "Soma realizada! O resultado foi:" e, logo em seguida, a expressão matemática soma as duas variáveis e já mostra o resultado para o usuário.

## 🚼 Atividade 3:

<img width="906" height="726" alt="image" src="https://github.com/user-attachments/assets/355f216b-b74e-4ea6-9382-21ec71f80344" />

A terceira atividade é um algoritmo que pede ao usuário sua idade e, por meio de uma condicional, diz se ele é maior ou menor de idade.
A primeira coisa a se fazer é criar uma variável de número inteiro onde será armazenado a idade do usuário. Por meio de um bloco de saída, o computador diz "Digite sua idade:" e então permite que o usuário digite um número inteiro. O número que for digitado será o novo valor da variável idade.
Em seguida, criamos uma condicional que verifica se idade >= 18 (Maior ou igual a 18). Caso essa condição seja verdadeira (seja maior ou igual a 18), ele diz que o usuário é maior de idade. Caso seja menor que 18, ele diz que o usuário é menor de idade.

## 🔢 Atividade 4:

<img width="951" height="685" alt="image" src="https://github.com/user-attachments/assets/34a1a79a-d082-4c83-b057-4d04a3c775c5" />

Na quarta atividade, foi criado um algoritmo que recebe um número e identifica se ele é positivo, negativo ou igual à zero.
Primeiro é definida a variável "numero" que pode armazenar um número real, que será o número analisado. Em seguida, o usuário pode digitar um número, e esse número vai definir o valor da variável. 
Depois, criamos uma condicional que vai verificar se o número é menor ou maior/igual que zero. Se for menor que zero, então o computador diz que o número é negativo. Porém, se o número for maior que zero, então uma segunda condicional é utilizada, que verifica se o número é igual a zero. Se for igual a zero, ele informa que o número é igual a zero. Caso não seja igual a zero, ele diz que o número é positivo.

## 🏫 Atividade 5:

<img width="362" height="665" alt="image" src="https://github.com/user-attachments/assets/80558bed-b064-4614-b183-80efe934d06d" />

Na quinta atividade, criamos um algoritmo que calcula a média de dois números reais. 
Primeiro, definimos duas variáveis de números reais, primeiraNota e segundaNota. Em seguida, o usuário define seus valores (que podem ser números quebrados) a partir de entradas de dados. 
Quando o usuário define os dois valores, uma saída de dados utiliza o seguinte código: "A média entre as duas notas foi: " & (primeiraNota + segundaNota) ÷ 2
Dessa forma, ele primeiro soma os dois números e só então os divide por dois.
